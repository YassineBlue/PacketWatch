# PacketWatch

PacketWatch is a lightweight PHP & MySQL telemetry dashboard for simulating and monitoring API request activity in real time through a clean Bootstrap interface.

## Features

- Real-time simulated request logging
- Request monitoring dashboard
- Live AJAX updates
- Authentication system
- Request status tracking
- Latency monitoring
- Search & filtering
- Responsive Bootstrap UI

## Tech Stack

- PHP
- MySQL
- Bootstrap 
- JavaScript
- AJAX
