# PacketWatch

PacketWatch is a simple PHP & MySQL dashboard for simulating and monitoring API request activity in real time through a clean Bootstrap interface.

## Features

* Simulated request logging
* Live request monitoring
* AJAX-based updates
* Authentication system
* Status and latency tracking
* Search and filtering
* Responsive Bootstrap UI

## Tech Stack

* PHP
* MySQL
* Bootstrap
* JavaScript
* AJAX

## Currently Working On

* Live charts
* Advanced filtering
* Export system
* WebSocket updates
* API analytics
