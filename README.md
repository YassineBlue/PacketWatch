# PacketWatch

PacketWatch is a lightweight PHP & MySQL telemetry dashboard for simulating and monitoring API request activity in real time through a clean Bootstrap interface.

## Features

- Real-time simulated request logging
- Request monitoring dashboard
- Live AJAX updates
- Authentication system
- Request status tracking
- Latency monitoring
- Search & filtering
- Responsive Bootstrap UI

## Tech Stack

- PHP
- MySQL
- Bootstrap 5
- JavaScript
- AJAX

```txt
packetwatch/

├── public/

```

## Installation

1. Clone the repository

```bash
git clone <repo-url>
```

2. Import the database

```sql
packetwatch.sql
```

3. Configure database credentials

```php
core/config.php
```

4. Start Apache & MySQL

5. Open the project in your browser
